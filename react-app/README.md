# Power BI JSON Theme Designer – React App

This is the main application. For setup instructions, see the [root README](../README.md).

## Development

```bash
npm install
npm run dev       # Start dev server at localhost:5173
npm run build     # Production build
npm run preview   # Preview production build
npm run lint      # ESLint check
```

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- Zustand (state management)
- MSAL (Azure AD authentication)
- Power BI Client React (embedded report)
