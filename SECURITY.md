# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest on `main` | ✅ |
| older commits | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do NOT open a public GitHub issue.**

Instead, please use one of the following:

1. **GitHub Private Vulnerability Reporting:**
   Go to the [Security Advisories](https://github.com/marlonetzel06/Power-Bi-JSON-Designer/security/advisories/new) page and create a new draft advisory.

2. **Email:**
   Contact the maintainer directly via their GitHub profile.

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline

- **Acknowledgement:** within 3 business days
- **Status update:** within 10 business days
- **Fix release:** as soon as possible, depending on severity

## Scope

This policy covers the Power BI JSON Theme Designer application code.
Third-party dependencies are handled through regular updates and audits.
