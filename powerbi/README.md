# Power BI Template File

Place the `All_visuals_template.pbix` file in this folder.

This report contains all supported Power BI visuals and is used as the live preview target for the Theme Designer app.

## Usage

1. Open the `.pbix` file in Power BI Desktop
2. Publish to your Power BI workspace
3. Copy the **Report ID** and **Workspace ID** from the URL
4. Add them to `react-app/.env.local`
